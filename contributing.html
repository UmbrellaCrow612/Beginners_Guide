Contributing to a project

Date: July 21, 2022
Author: Yousaf Wazir
Source: Tango

1. Go to the repository

2. Click on Fork

3. Give it a name

4. Click on Create fork

5. Click on Code

6. Click on Copy to clipboard

7. Make a folder

8. Open the folder in VsCode

9. In VsCode press command + J to open up the terminal

10. write "git clone https://github.com/UmbrellaCrow612/test-name.git ./" in the terminal and enter

11. Then run yarn in the terminal to install all the dependencies

12. Make your chnages

13. Then go to VsCode source controle

14. In source control press "+" in the changes you made

15. add a commit message then press commit

16. Then press sync changes

17. Go to the repo on github

18. Then go to pull request and create a new pull request

19. Then it shows you your changes then press create pull request

20. Then the project owner will review it and then merge it to the main project

This workflow was created with Tango.
