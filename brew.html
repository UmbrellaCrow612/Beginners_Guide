Install brew for MacOs

Date: July 21, 2022
Author: Yousaf Wazir
Source: Tango

This is for MacOs only

1. Go to brew install - Google Search

2. Click on main

3. Go to The Missing Package Manager for macOS (or Linux) — Homebrew

4. Click on /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"…

5. Paste this into your terminal, make sure your in your admin account
